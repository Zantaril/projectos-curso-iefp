
**An Integer to Binary, Hexadecimal or Octal converter**

  This is a small script I made to learn how the different number systems worked

**To run**

  Simply run the file on your terminal:
  
    python binHexConverter.py

**Example output**
  
  Choose an integer to convert into Octal.
  
  R: 511
  
  511 in Octal is:  777
