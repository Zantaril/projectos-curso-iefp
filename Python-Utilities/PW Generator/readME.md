**Password generator created as class exercise** 


This is a random password generator script that takes words from a text file and mixes
in symbols and numbers to then creat a password. 

**To run** 

 Make sure the text file (WordsList.txt) is present in the same folder as the script.
 
 Run the script in your terminal:
 
     python PwGenerator.py

**Example Output**

  Quantas palavras quer na sua password(1 a 10):2

  utilização2consciência@
 
  Enjoy your password!
